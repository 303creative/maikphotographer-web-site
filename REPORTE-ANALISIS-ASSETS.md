### ❌ PROBLEMAS ENCONTRADOS

1. **Optimización de imágenes JPG**
   - `38_miami_bg.jpg` = 6.9 MB (muy pesada para web)
   - `36_foto_camara.jpg` = 5.2 MB (muy pesada)
   - `45_scroll_video_ojo_de_cámara.mp4` = 16 MB (video muy pesado)
   - **Acción:** Comprimir a ~500-800 KB máximo
   - **Tool:** TinyPNG, ImageOptim, FFmpeg

2. **Videos sin optimización para web**
   - MP4 sin compresión moderna (H.265)
   - No hay formatos alternativos (WebM, AVIF)
   - **Acción:** Re-encode a H.264 con bitrate optimizado (~2-3 Mbps)
   - **Tool:** FFmpeg command: `ffmpeg -i input.mp4 -c:v libx264 -crf 23 output.mp4`

3. **Archivos Illustrator sin usar**
   - `Sin título-3.ai` = 260 KB sin utilidad en web
   - **Acción:** Eliminar o guardar en carpeta separada

4. **Texturas PNG muy grandes**
   - `30_textura_papel.png` = 3.4 MB 
   - **Acción:** Exportar más pequeño o usar SVG pattern
   - **Alternativa:** Usar CSS generado para noise/texture

5. **Color branding incompleto**
   - Assets mezclan gold (#D8C18A) con naranja (#FF5722)
   - El design-system.css usa #FF5722, assets usan #D8C18A
   - **Acción:** Unificar colores antes de integrar

6. **Falta responsive para móvil**
   - Videos no tienen versión mobile (más pequeña)
   - Imágenes hero 5MB+ cargarán lento en 4G
   - **Acción:** Crear media query con srcset

### ⚠️ ADVERTENCIAS TÉCNICAS

1. **Hero videos vs Performance**
   - Agregar 40-45 MB de video ralentizará página
   - Solución: Lazy-load videos, cargar solo en desktop
   - Usar `<source>` con media queries

2. **Animaciones GSAP + Scroll videos**
   - `animations.js` ya usa ScrollTrigger
   - Los videos scroll-trigger pueden conflictuar
   - Necesita coordinación en `animations.js`

3. **Watermarks con canvas**
   - Si agregas watermarks dinámicos, asegura no bloquear gallery lightbox
   - Considera performance en móvil

### Lo que se puede animar con nuevos assets:

**Videos con scroll:**
```javascript
gsap.to(videoElement, {
  currentTime: 16, // play 16 seconds of video
  scrollTrigger: {
    trigger: videoElement,
    start: 'top 80%',
    end: 'center center',
    scrub: 1
  }
})
```

**Parallax con fondos:**
```javascript
gsap.to('.hero-bg', {
  yPercent: 50,
  ease: 'none',
  scrollTrigger: {
    trigger: '.hero',
    start: 'top top',
    end: 'bottom top',
    scrub: true
  }
})
```

**Fade overlay texturas:**
```javascript
gsap.fromTo('.grain-overlay', {
  opacity: 0.02
}, {
  opacity: 0.08,
  scrollTrigger: {
    trigger: '.portfolio',
    start: 'top 80%'
  }
})
```
