# Photography Skill — Technical & Creative Standards

## Objetivo
Guiar agentes en selección, edición y presentación de fotografía.

---

## 1. PHOTOGRAPHY STYLE

### Maikel Marshall Aesthetic
- **Cinematic**: Lighting like film (no flash, natural/golden)
- **Color-graded**: Warm, film-like color palette
- **Shallow depth**: 50-85mm lenses, f/2.8-f/4
- **Composition**: Rule of thirds + negative space
- **Story-driven**: Each image has context, emotion
- **Filmic**: Feels like stills from a movie

### Visual Language
```
✓ Warm color temps (3200-4500K)
✓ Shallow focus (subject isolated)
✓ Natural light or golden hour
✓ Minimal but intentional composition
✓ Skin tones: warm, not flat
✓ Shadows: visible but not crushed
✓ Highlights: detailed, not blown
```

---

## 2. EDITING STANDARDS

### Color Grading
```
Baseline:
- Temp: 3200-3800K (warm, not orange)
- Tint: -5 to +5 (slight cyan for depth)
- Saturation: +5-15 (vibrant not oversaturated)

Skin Tones:
- Yellow/Green: Adjust until skin looks alive
- Magenta/Green: Balance for natural tone
- Target: Warm but not washed out

Shadows:
- Lift: +5-15 (detail visible, not crushed)
- Adjustment: Lift shadows WITHOUT losing contrast

Highlights:
- Recovery: +50-80 (preserve detail in blown areas)
- Clipping: None visible on skin or whites
```

### Post-Processing
```
Sharpness:      +15-25 (crisp not artificial)
Clarity:        +10-20 (pop without halo)
Vibrance:       +10-15 (life without over-saturation)
Highlights:     -10-0 (preserve detail)
Shadows:        +10-20 (open without crushing blacks)
Whites:         -5-5 (balance exposure)
Blacks:         -5-10 (punch without crush)
```

### Avoid ❌
```
- Crushed blacks (no detail)
- Blown highlights (flat skin)
- Over-vibrant colors (artificial)
- Excessive blur (unnatural DoF)
- Lens corrections too obvious
- Excessive clarity (weird halos)
- Mismatched white balance
```

## 3. PORTFOLIO CURATION RULES

### Selection Criteria
```
Technical:
✓ Perfect focus on subject
✓ Proper exposure (no clipping)
✓ Sharp, not soft or motion-blurred
✓ Good composition (intentional, not accidental)

Creative:
✓ Tells a story (emotion visible)
✓ Unique perspective (not generic)
✓ Color harmony (intentional palette)
✓ Strong subject (clear what's important)

Professional:
✓ Consistent editing style
✓ No duplicates or near-duplicates
✓ Lighting is professional (no harsh shadows)
✓ Matching brand aesthetic
```

### Reject Criteria ❌
```
- Soft focus (even slightly out)
- Blown highlights on face
- Crushed shadows (no detail)
- Distracting background (pulls attention)
- Poor composition (subject lost)
- Over-processed (artificial look)
- Inconsistent with brand style
- Test shots or work-in-progress
```

## 5. LIGHTING SETUPS

### Golden Hour (Preferred)
```
Time:       1 hour before sunset
Quality:    Warm, directional, flattering
Direction:  Side-light or backlight
Intensity:  Medium (not harsh)
Skin tone:  Golden, warm, natural
Best for:   Editorial, lifestyle, couples
```

### Overcast Day
```
Condition:  Cloudy, no direct sun
Quality:    Soft, even, shadow-free
Direction:  Omnidirectional
Intensity:  Consistent
Skin tone:  Cool but clear
Best for:   Portrait, close-ups, details
```

### Studio Lighting
```
Key light:     Soft modifier (octabox 5ft+)
Fill light:    Reflector or 2nd light (30% intensity)
Backlight:     Rim light for separation (optional)
Intensity:     f/2.8-f/4 range
Modifiers:     Softboxes, beauty dishes (not bare)
```

### Night/Event
```
Ambient:       Use available light when possible
Flash:         Bounce or diffuse (no harsh direct)
ISO:           Push to 3200-6400 if needed
Shutter:       1/60-1/125 (camera shake prevention)
Focus:         Back-button AF for accuracy
```

## 6. LENS CHOICES

### Portrait (55-85mm)
```
Focal:         55-85mm
Aperture:      f/1.8-f/2.8
Distance:      6-10 feet from subject
Background:    Soft, out of focus (shallow DoF)
Use:           Headshots, close-up portraits
```

### Editorial (35-50mm)
```
Focal:         35-50mm
Aperture:      f/2.8-f/4
Distance:      10-15 feet
Background:    Visible but soft
Use:           Environmental portraits, context
```

### Wide (24-35mm)
```
Focal:         24-35mm
Aperture:      f/2.8-f/4
Distance:      15+ feet
Background:    Fully visible
Use:           Location shoots, story context
```

## 7. COMPOSITION TECHNIQUES

### Rule of Thirds
```
Horizon line:    1/3 or 2/3 from top (not centered)
Subject:         1/3 from side (not centered)
Eyes:            Upper third (psychological)
Hands:           Frame edge or full visible
```

### Leading Lines
```
Use:       Paths, horizons, arm direction
Purpose:   Draw eye to subject
Avoid:     Lines that cut subject in half
Example:   Leading lines toward face
```

### Depth Layers
```
Foreground:   Optional context
Subject:      Sharp, focused
Background:   Soft, blurred
Result:       3D feeling, not flat
```

### Negative Space
```
Use:          Give breathing room
Amount:       1/3 empty space minimum
Purpose:      Make subject feel less crowded
Avoid:        Subject crammed in corner
```

- [ ] White balance (auto or custom?)
- [ ] Exposure compensation (scene brightness?)

## 9. EDITING WORKFLOW

1. **Import & Organize**
   - Import RAW files
   - Create backup
   - Rate keepers (1-star minimum)

2. **White Balance**
   - Custom white balance per lighting setup
   - Match skin tone (warm not orange)

3. **Exposure & Tonality**
   - Set blacks (Shadows slider +10-20)
   - Set whites (Highlights slider -10-0)
   - Adjust midtones (Exposure ±0.5)

4. **Color Grade**
   - Temp: 3200-3800K
   - Tint: -5 to +5
   - Saturation: +5-15

5. **Detail**
   - Sharpness: +15-25
   - Clarity: +10-20
   - Vibrance: +10-15

6. **Finalize**
   - Export JPG (quality 75-80)
   - Review at 100% zoom
   - Compare to edited batch (consistency?)

## 12. COMMON ISSUES & FIXES

### Soft Focus
- Problem: Images not sharp
- Cause: Slow shutter, focus miss
- Fix: Faster shutter (1/125+), back-button AF, burst mode

### Blown Highlights
- Problem: Blown skin/whites
- Cause: Metering to subject, bright background
- Fix: Expose to right, recover in post, use ND filter

### Crushed Blacks
- Problem: No shadow detail
- Cause: Under-exposure, aggressive editing
- Fix: Lift shadows in post, expose properly in-camera

### Inconsistent Editing
- Problem: Images look different from each other
- Cause: Different lighting, missing presets
- Fix: Create LUT presets, match batch consistently

### Flat Lighting
- Problem: No dimension, boring
- Cause: Flat overcast or direct sun
- Fix: Use golden hour, add fill light, use reflector
